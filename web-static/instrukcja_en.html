Instruction of pool

We select the appropriate version cgminera / bfgminera / guiminera (for manual we will use cgminer)
If you have a rig with module and built cgminer go to the extra point *

1 Depending on whether you have a hardware CPU, GPU, FPGA, ASIC Here we choose this software:

CPU, GPU, FPGA - promoted to version 3.7.2 cgminera (later discontinued in Sparta for the gpu, cpu)
Github: https://github.com/Kalroth/cgminer-3.7.2-kalroth
Done: http://k-dev.net/cgminer/

ASIC - supporting the latest cgminerami
Github: https://github.com/ckolivas/cgminer
Done: http://ck.kolivas.org/apps/cgminer/

2 The relevant software depending on what we have excavator:

Processor Processor driver, mandatory control with Open CL

Radeon / NVidia cards , Download the appropriate drivers from the manufacturer's sites, mandatory control with Open CL
Radeon - http://support.amd.com/en-us/download
NVidia - http://www.nvidia.pl/Download/index.aspx?lang=pl

FPGA Dedicated drivers depend on the type of FPGA, usually the manufacturer's website:
ztex - http://www.ztex.de/downloads/#firmware_kit

ASIC To use cgminera in most cases we will need drivers WinUSB
We can download it from this page: http://zadig.akeo.ie/

3 Now, wanting to run an excavator poolu we need to create an executable batch.
I will explain step by step how to do it:
-open notebook -passed this "cgminer api.thash.pl:9332 o u p YourBTCAddress xxx"
-Saves file as start.bat in folder of cgminer
(remember to select Save as type: all files)
